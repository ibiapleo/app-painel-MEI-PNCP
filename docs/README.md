## Como usar

Cada pasta de capstone contém:

- `README.md` — visão geral do capstone (escopo definido pelo professor, entregas, status, divisão de atividades).
- Documentos específicos da entrega (contratos de API, arquitetura, decisões técnicas, protótipos, testes, retrospectiva, etc.).

Para adicionar uma nova documentação, crie o arquivo `.md` dentro da pasta do capstone correspondente e referencie-o no `README.md` daquela pasta.
